# Самостійна робота

## Узагальнений кеш `Cache<T>` із обмеженнями, очищенням та власним сортуванням

### Мета роботи

- Закріпити знання з **узагальненого програмування (generics)** у C#.
- Навчитись застосовувати **обмеження типів (constraints)**: `class`, `struct`, `new()`.
- Реалізувати **очищення старих елементів кешу** (TTL — time-to-live).
- Розробити **власний алгоритм сортування** без використання `List.Sort()` або LINQ.

---

### Теоретичні відомості

Узагальнені класи C# дозволяють описувати алгоритми, не прив’язані до конкретного типу даних.

**Типові обмеження:**

```csharp
where T : class          // лише посилальні типи
where T : struct         // лише значимі типи
where T : new()          // тип має публічний конструктор без параметрів
```
